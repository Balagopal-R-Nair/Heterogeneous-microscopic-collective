This repository contains code for reproducing the results from, ”Hydrodynamic interactions mask the true heterogeneity of a microscopic collective”, by Balagopal Nair, Arshed Nabeel, and Danny Raj M.

The scripts model_functions.py contains the main model simulation code and model_run.py is used to run simulation. The model_run.py file can be edited to change the initial parameters. The only ones that are varied in the paper are If and sigma. The file can also be edited to decide whether to run the simulation for one instance or an ensemble of realizations by commenting out the neccesary function.

Each script in Figures can recreate the figures from the paper. 
For figures 2 and 3 original data files are too large so the simulation needs be run for each set of parameters. In model_run.py, run the run_model_ensemble function with each pair of initial parameters If and sigma. The output files that obtained will have the same name as the file names that are already loaded in the scripts for figures 2 and 3.

Figures 4,5 and 6b have the original data files uploaded along with the script. The data for figures 4 and 6b can also be re generated by uncommenting lines 503 and 506 in model_functions.py.
