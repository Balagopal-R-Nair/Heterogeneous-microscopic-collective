This repository contains code for reproducing the results from, ”Hydrodynamic interactions mask the true heterogeneity of a microscopic collective”, by Balagopal Nair, Arshed Nabeel, and Danny Raj M.
Each script in Figures can recreate the figures from the paper. Figures 4,5 and 6b have the original data files. For figures 2 and 3 original data files are too large. The simulation can be run for each set of parameters by uncommenting run_model_ensemble in model_run.py to get the required data. The main model simulation code is in model_functions.py

The data for figures 4 and 6b can also be re generated by uncommenting the lines in model_functions.py.
