Each script in Figures can recreate the figures from the paper. Figures 4,5 and 6b have the original data files. For figures 2 and 3 original data files are too large. The simulation can be run for each set of parameters by uncommenting run_model_ensemble in model_run to get the required data.

The data for figures 4 and 6b can also be re generated by uncommenting the lines in model_functions.
